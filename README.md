# Курсовая работа по базам данных

## Описание проекта
Проект для получения данных о компаниях и вакансиях с сайта hh.ru и загрузки их в базу данных PostgreSQL.

## Функциональность
- Получение данных о работодателях и их вакансиях с hh.ru
- Сохранение данных в базу данных PostgreSQL
- Анализ вакансий через класс DBManager

## Установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте базу данных PostgreSQL
4. Настройте параметры подключения в файле `.env`

## Использование
Запустите main.py для начала работы с программой.
